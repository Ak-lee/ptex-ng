## How to Build (Linux/Mac)

Run the shell script:

    ./build-pTeX-ng.sh

### Building Chain

* zlib
* libpng
* libpaper
* kpathsea
* ptexenc
* libdpx
* ptex-ng

## How to Build (Android)

    ./build-pTeX-ng.sh
    cd android
    ./build-android-port.sh
